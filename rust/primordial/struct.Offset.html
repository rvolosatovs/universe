#[repr(transparent)]
pub struct Offset<T, U>(_, _);
Expand description

An offset of a number of items of type T from a base

Note well that this is NOT stored in memory as the number of bytes, but rather the number of items.

One important additional feature is that offsets can be converted between underlying types so long as the conversion is lossless for the target CPU architecture. For example, Offset<u64> can be converted to Offset<usize> on 64-bit systems.
