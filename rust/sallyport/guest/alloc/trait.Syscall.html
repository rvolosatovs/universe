type Staged: Commit<Item = Self::Committed>

Opaque staged value value, which returns Self::Committed when committed via Commit::commit.

This is primarily designed to serve as a container for dynamic data allocated within stage.

[src]

type Committed

Opaque committed value returned by Commit::commit called upon Self::Staged, which is, in turn, passed to Self::collect to yield a Self::Collected.

[src]

type Collected

Value syscall collects as, which corresponds to its return value.
